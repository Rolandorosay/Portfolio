PacMen Exercise

This is a factory for making PacMen. When you hit the "Add PacMan" button another PacMan will appear on the screen. To start the game and have the PacMen running around the screen just hit the "Start Game" button. The exercise was created to show how an image can change and run across the screen in random directions and at random speeds. PacMan will bounce around chomping. (still working on getting him to chomp and switch directions)

Quick Bio

Hey! My name is Rolando Rodriguez and I am an architectural designer and self-taught UI/UX designer. I am learning to code so that I can bring the skills I learned in UI/UX full circle. I am interested in learning how to build apps and websites so that I can build working prototypes before bringing on friends to work with. I enjoy thinking of new ways to solve problems, and I think code will help me be better equipped to do so. Learning code will also open up new opportunities in other interests of mine like machine learning. Using a robotic arm to 3D print furniture or architectural details is just the beginning of it. There are so many possibilities when it comes to machine learning! I'd also like to mention the coding community. Those that I have experienced so far are very involved. Everyone is helping each other and trying to figure out things together. It can be frustrating at times sitting in front of the computer for hours trying to figure out one little aspect needed to elevate your project to the next level, but the community has been quite helpful. Moving forward, I am excited to see all the opportunities learning code brings with it.
